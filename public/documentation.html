MyFilm

Where users can learn about my favorite movies


The following table describes the endpoints for the website

Description URL HTTP Method Request Body Data Format Response Body Data Format
Display all movies /movies GET none JSON object of all movies
Display information of a specific movie /movies:title GET none JSON object of movie data- title, director, genre
Describe a chosen genre /genre/:name GET none JSON object containing description of chosen genre
Display director bio /director/:name GET none JSON object of name, birth year, nationality, and death year of director
New user register /users POST JSON object with Username, password, email, and DOB JSON object displaying all of the information just added
User updates their profile /users/:username/:password/:email/:dob PUT None Message of confirmation that it was updated
User adds a movie to their list of favorites /users/:username/favorites/:title POST none Message confirming the movie was added
User can delete a movie from their favorite list /users/:username/favorites/:title DELETE None Message confirming the movie was removed
User can deregister /users/:username DELETE None Messsage confirming they are removed
